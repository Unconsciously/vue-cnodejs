# vue-cnodejs

> 基于vueJS/vuex/vue-router/axios/scss/mint-ui/mavon-editor/flexible/webpack/重写Cnodejs.org社区

技术栈：
- vueJS
- vuex 
- vue-router
- scss
- axios 
- mint-ui 
- scss
- mavon-editor
- lib-flexible
- webpack


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
